# physics-quiz
Physics Quiz App ⚡📊
A command-line Python quiz application designed to test physics concepts and analyze topic-wise preparation levels. The program asks multiple questions, calculates the score, and visualizes performance using a bar graph.
🚀 Features
Multiple-choice and input-based questions
Score calculation (/10)
Topic-wise performance tracking
Data visualization using Matplotlib
📊 Output
Displays final score
Shows a bar graph representing preparation level across topics
